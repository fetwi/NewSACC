<h3 class="clause-heading">Legal text for SACC item</h3><pre><div class="sacc-item">
<p>The Contractor must apply bar code information on items
<span class="input-required"><var>_______</var> <kbd>(<strong><em>insert the list</em></strong>)</kbd></span> with the Permanent System Control
Number (PSCN) or <abbr title="North Atlantic Treaty Organization">NATO</abbr> Stock Number (NSN) provided elsewhere in the
Contract or by the Department of National Defence (DND), using bar
code standard <abbr title="Uniform Code Council">UCC</abbr>/<abbr title="European Article Numbering">EAN</abbr>-128 (Uniform Code Council/EAN International)
with Application Identifier 241 for <abbr title="Permanent System Control Number">PSCN</abbr> or 7001 for <abbr title="NATO Stock Number">NSN</abbr>. Below the
bar code symbol, the Contractor must apply the Human-Readable
Interpretation (HRI) markings.</p>
<p>These markings must be applied and positioned in accordance with
Department of National Defence standard D-02-002-001/SG-001,
Identification Marking Of Canadian Military Property (in effect at
the closing date of the bid solicitation), and must be of such
quality that it will remain readable for the expected life of the
item. The bar code must be imprinted upon material which will be
compatible with the item to which it is to be attached, which items
include, but are not limited to, items constructed of plastic,
metal, cloth, synthetics or paper, or a combination of two or more
of them.</p>
